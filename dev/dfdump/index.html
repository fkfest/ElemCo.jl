Generation of integrals using density fitting

ElemCo.DfDump — Module

generate fcidump using df integrals and store in dumpfile

source

Exported functions

ElemCo.DfDump.dfdump — Method
dfdump(EC::ECInfo)

Generate fcidump using df integrals and store in IntOptions.fcidump. If IntOptions.fcidump is empty, don't write to fcidump file, store in EC.fd.

source

Internal functions

ElemCo.DfDump.generate_integrals — Method
generate_integrals(EC::ECInfo, fdump::TFDump, cMO::SpinMatrix, full_spaces)

Generate int2aa, int2bb, int2ab, int1a, int1b and int0 integrals for fcidump using density fitting.

mpfit basis is used for int2 integrals, and jkfit basis-correction is used for int1 and int0 integrals. full_spaces is a dictionary with spaces without frozen orbitals.
