ElemCo.CCTools.starting_amplitudes — Method
starting_amplitudes(EC::ECInfo, method::ECMethod)

Prepare starting amplitudes for coupled cluster calculation.

The starting amplitudes are read from files T_vo, T_VO, T_vvoo, etc. If the files do not exist, the amplitudes are initialized to zero. The order of amplitudes is as follows:

  • singles: α, β
  • doubles: αα, ββ, αβ
  • triples: ααα, βββ, ααβ, αββ

Return a list of vectors of starting amplitudes and a list of ranges for excitation levels.
